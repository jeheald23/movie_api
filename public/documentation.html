MyFlix App Documentation

A server-side movies application which will provide users with access to information about different movies, directors, and genres. Users can sign up, update their personal information, and create a list of their favorite movies.

USERS

Business Logic URL HTTP Method Request Body Data Format Response Body Data Format
Get a list of all users /users GET None A JSON array of objects.
Register a user /users POST A JSON object holding data with the user information to add, including name, username, and favorites. For example,

{

Name: "Fred Jones",

Username: "fredjones",

Favorites:[

Mulholland Drive,

8 1/2

]

}

 A JSON object holding data about the user that was added including an ID. For example,

{

ID: 2,

Name: "Fred Jones",

Username: "fredjones",

Favorites:[

Mulholland Drive,

8 1/2

]

}
Get data about a single user by username /users/:username GET None A JSON object holding data about a single user, containing an ID, name, user name, and favorites. For example,

{

ID: 1,

Name: "Bob Smith",

Username: "bobsmith",

Favorites:[

Blade Runner,

A Clockwork Orange,

Stalker

]

}
Update a user's info /users/:username PUT None A text message indicating whether the user's info was updated.
Add a movie to a user's list of favorites /users/:username/favorites POST None A text message indicating whether a movie was added to the user's favorites.
Get a user's list of favorites /users/:username/favorites GET None A JSON array of objects.
Remove a movie from a user's list of favorites /users/:username/favorites DELETE None A text message indicating whether a movie was deleted from the user's favorites.
Remove a user by ID /users/:userID DELETE None A text message indicating whether the user was deleted.

MOVIES

Business Logic URL HTTP Method Request Body Data Format Response Body Data Format
Get a list of all movies /movies GET None A JSON array of objects.
Get a movie by title /movies/:title GET None A JSON object holding data about a single movie including the title, description, genre, director, image URL, and whether it’s featured or not. For example,

{

Title: "Blade Runner",

Director: "Ridley Scott",

Genre: "Science Fiction",

Description: "The film is set in a dystopian future Los Angeles of 2019, in which synthetic humans known as replicants are bio-engineered by the powerful Tyrell Corporation to work on space colonies. When a fugitive group of advanced replicants led by Roy Batty escapes back to Earth, burnt-out cop Rick Deckard reluctantly agrees to hunt them down. (from Wikipedia)",

Image URL: "https://en.wikipedia.org/wiki/Blade_Runner#/media/File:Blade_Runner_(1982_poster).png",

Featured: "True"

}
Get a movie by director /movies/:director GET None A JSON array of movies by director. For example,

{

Director: "David Lynch",

Movies: [

Mulholland Drive,

Eraserhead,

Blue Velvet

]

}
Get a list of movies by genre /movies/:genre GET None A JSON array of movies by genre. For example,

{

Genre: "Science Fiction",

Movies: [

Blade Runner,

A Clockwork Orange,

Stalker

]

}
Get the genre of a movie by name /movies/:title/genre GET None A JSON object holding the genre of a movie. For example,

Movie: "Blade Runner", Genre: "Science Fiction"
