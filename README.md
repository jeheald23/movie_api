# MyFlix App Documentation

A server-side movies application which provides users with access to information about different movies, directors, and genres. Users can sign up, update their personal information, and create a list of their favorite movies.

## USERS

| Business Logic                              | URL                              | HTTP Method | Request Body Data Format                                                                                                                                                     | Response Body Data Format                                                                                                             |
|---------------------------------------------|----------------------------------|-------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------|
| Allow a new user to register                | /users                           | POST        | A JSON object holding data with the user information to add including username, password, and birthday. For example,<br>{<br>"Username": "plato",<br>"Password": "socrates1234",<br>"Email": "plato@academy.com",<br>"Birthday": "1954-01-02"<br>}    | A JSON object holding data about the user that was added including an ID and an empty array for FavoriteMovies to be added. | Allow a user to login                       | /login                           | POST        | User must enter username and password key and username and password value to query params.
